## Features

- [x] Basic UI / views

## Stack used

- Alamofire `4.5`
- SwiftyJSON `4.0`
- Cocoapods `1.3.1`
- Swift `3`

## Getting started

#### Requirements

- Xcode `8.3.2`
- CocoaPods (`pod` command)

#### Installation

```sh
git clone https://github.com/jclerc/learning-ios.git
cd learning-ios
pod install
open -a Xcode AppCours.xcworkspace
```
